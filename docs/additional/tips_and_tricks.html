Tips and Tricks¶

Jupyter Notebook keyboard shortcuts¶

  • Everything is a cell

  • cells are either code or md

  • there are two modes - edit or command

  • ESC - Command mode

  • Enter - edit mode

  • A - insert above

  • B - insert below

  • DD - delete

  • M - md

  • Y - code
